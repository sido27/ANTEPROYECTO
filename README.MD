# Título : Spain Travels 

# Autor : Asid kouki (Sido) 

# Introdución : 
***Spain Travels*** será un wesbite para una agencia de viajes/turismo.
está dirigido a los entusiastas de los viajes en España que quieren descubrir 
las bellezas de la naturaleza ya sea en España o en cualquier otro país del mundo , 
***Spain Travels*** organiza viajes de diferentes tipos para satisfacer todas las necesidades 
de los viajeros y para asegurarse de que sus curiosidades sean satisfechas.


# Finalidad :
Este sitio web ofrece un portal para las personas que tienen ganas de viajar para ver 
la belleza de la naturaleza que el mundo tiene que ofrecer y darles a conocer nuevos lugares que de otro modo serían desconocidos.


# Objetivo :

El sitio web incluirá diferentes tipos de tours/viajes basados en la ubicación:
(local o internacional) y la actividad como el senderismo, la escalada, 
el descubrimiento de monumentos y ruinas antiguas, 
los viajes a la selva e incluso las actividades subacuáticas, 
todos los cuales son calificados por los viajeros que también pueden subir imágenes de su viaje y compartirlo con el mundo, 
así como comentar sus pensamientos para que otros los lean.

Una vez que un visitante del sitio web decida participar en un viaje con nosotros, se le pedirá que se autentique para poder reservar su viaje.
